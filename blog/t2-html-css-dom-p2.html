HTML CSS & DOM part 2

14.Feb.2018

What is Grid base design?

  • Grid base design is the framework and structure of a webpage html and css. It's when you structure
    the page into columns and rows for a better and easier to work with page layout.
Whats all the hype about responsive webpages?
  • Responsive webpages are ideal for your page as it caters towards the device you are viewing it on.
    a responsive webpage will change its layout to best fit your screen and give the user the best experience.
What is semantic structure?
  • Semantic structure is when you structure your elements from important to less important, and applying the best
    semantic tags to emphasize its importantness in relation to less important things
