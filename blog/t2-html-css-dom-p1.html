Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

What is meant by boxifying design?
  • boxifying is when you visulise the elements of the website design into boxes to make the layout more clear
What is the box model?
  • Box model is the visual representation of the space and areas of and around the element eg. content,padding,border,margin.
